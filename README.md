## Weather App
 A simple and modern Weather App built using HTML, CSS, and JavaScript.

 This app fetches real-time weather data using a public weather API and displays temperature, humidity, wind speed, and weather conditions for any city.

## Features
 Search weather by city name
 Real-time weather data
 Temperature, humidity, wind speed
 Weather condition icons (sun, clouds, rain, etc.)
 Error handling (invalid city)
 Responsive UI for mobile & desktop
 Clean and simple design


## Tech Stack
 CSS
 JavaScript
 OpenWeatherMap API

 Enter a city name in the search bar
 JavaScript sends a request to the Weather API
 API returns weather details
